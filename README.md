# Cosmic Landing (Vite + React + Tailwind)

This is a minimal starter project containing the **Cosmic Landing** page you requested:
- Animated nebula background
- Moving stars
- Golden cursor aura that follows the mouse
- Portfolio cards
- Big WhatsApp CTA button with template message

## Quick start

1. Install dependencies
   ```bash
   npm install
   ```
2. Run dev server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```
4. Preview production build
   ```bash
   npm run preview
   ```

## Notes
- Replace the placeholder images in `public/` with your own portfolio screenshots.
- Update the WhatsApp number inside `src/App.jsx` (search for `WA_NUMBER`).
- This project uses Vite (fast dev server) and TailwindCSS for styling.
